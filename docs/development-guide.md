## 추천 개발 순서

1. MVP 기능 목록 확정
2. 화면 와이어프레임 작성
3. 데이터 스키마 정의
4. 더미 데이터 기반 대시보드 구현
5. 쾌적도 계산 로직 연결
6. 전력/추천 로직 추가
7. 센서 또는 외부 데이터 연동

## Definition of Done

작업 완료 기준:

- 기능이 실행된다.
- README 또는 관련 문서가 업데이트됐다.
- PR 설명에 테스트/확인 방법이 있다.
- 다음 작업자가 이어받을 수 있다.
