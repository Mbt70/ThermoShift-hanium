# ThermoShift Project Brief

## 1. 한 문장 설명

중소형 공간에서 쾌적도와 전력 사용량을 함께 고려해 냉난방 운영을 추천하는 의사결정 보조 시스템.

## 2. 대상 사용자

- 소규모 사무실/스터디룸/강의실 관리자
- 실내 환경을 개선하고 싶은 공간 운영자
- 에너지 사용량을 줄이고 싶은 시설 담당자

## 3. 해결하려는 문제

- 사람의 체감 쾌적도는 온도 하나로 설명하기 어렵다.
- 냉난방을 과하게 쓰면 전력 사용량이 커진다.
- 반대로 절전만 추구하면 불쾌감이 커진다.
- 소규모 공간은 고가 BEMS 없이도 쓸 수 있는 가벼운 도구가 필요하다.

## 4. MVP 범위 초안

### Must

- 환경 데이터 입력/저장
- 쾌적도 점수 계산 또는 추정
- 현재 상태 대시보드
- 간단한 운영 추천 문구

### Should

- 히스토리 그래프
- 전력 사용량 추정
- 추천 전후 비교

### Could

- 사용자 피드백 반영
- 외부 날씨 데이터 연동
- 최적화 기반 제어 시나리오

### Won't for MVP

- 실제 냉난방기 자동 제어
- 대규모 상용 BEMS 수준 기능
- 민감 개인정보 수집

## 5. 성공 기준 초안

- 팀원이 같은 구조로 개발할 수 있다.
- 데모 화면에서 현재 상태와 추천 액션을 설명할 수 있다.
- 데이터 기반으로 쾌적도/전력 절감 논리를 제시할 수 있다.
- 최종 발표/보고서에 재사용 가능한 기록이 남는다.
